# Prerequisites
- A Linux system with recent NVIDIA drivers.
- Install the [CUDA toolkit](https://developer.nvidia.com/cuda-downloads).
- [nvCOMP](https://developer.nvidia.com/nvcomp-download) for Deflate decompression support
